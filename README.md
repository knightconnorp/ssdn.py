# ssdn.py
ssdn.py - Stupid Simple Directory Navigator

## Easy Integration
- VIM Keybinds for regular integration into workflow
- 'Q' or 'ESC' to quit
- Allows you to pivot directly into VI, VIM, or NVIM seamlessly
